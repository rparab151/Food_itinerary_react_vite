# Food Itinerary Builder (React + Vite)

## Local dev
1) Install Node.js 18+
2) npm install
3) Create .env from .env.example and set VITE_GOOGLE_MAPS_BROWSER_KEY
4) npm run dev

## Deploy on Vercel
- Import this repo to Vercel
- Set env var GOOGLE_MAPS_API_KEY in Vercel Project Settings (server key)
- Set env var VITE_GOOGLE_MAPS_BROWSER_KEY (browser key)
